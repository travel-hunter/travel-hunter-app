# Backend

FastAPI 기반 백엔드 API 서버

## 기술 스택
- Python
- FastAPI
- PostgreSQL (RDS)

## MVP API 설계

1차 MVP에서는 구현보다 API 계약을 먼저 고정합니다.

- 인증: 회원가입, 로그인
- 정책: 목록 조회, 상세 조회
- 일정: 목록 조회, 생성, 상세 조회
- 일정 장소: 추가, 삭제

상세 계약은 [`docs/mvp-api-contract.md`](../docs/mvp-api-contract.md)를 참고하세요.
